Клиенты и счета
Описание проекта:
Есть данные о банковских операциях, которые представлены в виде четырех таблиц. Формат хранения данных — сsv.

Используемый стек:
Первоначально тестировал локально на операционной системе Windows. Развернул локально Jupyter Notebook, для scala использовал ядро spylon-kernel, spark 3.3.1 локально. Но из-за проблем работы spylon-kernel c кодировкой UTF8 пришлось отказаться от данного варианта.
Далее развернул виртуальную машину на ОС Ubuntu 22:
Spark: 3.3.1
Scala: 2.12.15
Postgres: 15 версия
java: 1.8.0_352 (Private Build) 
Средстство разработки jupyter notebook 

Поскольку данный проект не предполагает потоковую обработку данных и данные представлены в виде статических таблиц, которые не пополняются, то в качестве решения была предложена локальная обработка. По схеме: загрузил->произвел преобразования->выгрузил. 
Также данные были загружены в БД с которой также данные считывались.
